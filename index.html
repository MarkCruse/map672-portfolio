Web Cartography at the University of Kentucky


St. Petersburg, FL bus stops with half mile buffer radius

The map depicts the location of each bus stop in the St. Petersburg city limits. A half mile buffer (yellow area) shows the walking distance to/from each stop.

The map was created using QGIS. Map data was obtained from OpenStreetMap (OSM) using the QuickOSM plugin for QGIS.

Fall 2016 Prelimary Enrollment - University of KY

Preliminary enrollment for the Fall 2016 undergraduate class. Enrollment totals per county are reflective of student reported county and state residency.

Data sources: University of Kentucky SAP/HANA enrollment tables. Note: Only aggregated county and state student information was extracted. Enrollment totals are reflective of student reported county and state residency. 2010 Census and 2015 Estimated Census data acquired online from Fact Finder Census.

Tools: The map layers were created using QGIS. The map layers were exported to Carto for assembly and map presentation.
